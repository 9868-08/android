
УСТАНОВКА.

на т.ф. зайти в режим fastboot (Pow + Vol-)
подключить т.ф. к п.к.

на п.к. зайти в диспетчер устройств (клик прав. кн. мыши по Мой компьютер -> Свойства -> Диспетчер устройств)

находим в "Другие устройства" -> Android -> клик по нему прав. кн. мыши -> выбираем "Обновить драйверы".

затем -> "Выполнить поиск дрв на этом п.к." -> "Выбрать дрв. из списка уже уст." -> клик "Далее" -> "Установить с диска"

в откр. окне -> "Обзор" -> найти, куда разархивировали файл и указать папку "Android_Composite_ADB..." -> "Открыть"

в след окне -> "ОК" -> выберете драйвер для этого устройства -> "Далее" -> пропускаем предупреждение клик "Да"
